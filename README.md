# Stairway to Heaven / 天堂阶梯

[![NeoForge](https://img.shields.io/badge/NeoForge-1.21.1-orange)](https://neoforged.net/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**把升级模块装入天堂之靴，经验解锁槽位，无需跳跃自动走上高墙。**

Fill Heaven Boots with Upgrade Molds, unlock slots via XP, and walk up walls without jumping.

---

## 功能 Features

- **天堂之靴** — 放在背包任意位置即可生效，根据升级等级自动上坡
- **升级 GUI** — 9 个槽位，每 10 级经验解锁一格（10→90 级）
- **升级模块** — 放入已解锁槽位提升等级，模具持久存储不消失
- **自动上坡附魔** — 最大 1 级，附在普通靴子上 +1 步高
- **摔落减免** — 随等级自动提高安全摔落高度
- **零依赖** — 仅需 NeoForge，兼容拔刀剑等模组

## 合成 Recipes

| 物品 | 配方 |
|------|------|
